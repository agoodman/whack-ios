h1. Whack

Anger-driven helpdesk triggering for iOS. When users shake or smack their device in frustration, soothe their concerns with a friendly tutorial view.

h1. Installation

Add this repo as a submodule. Then, add the Whack directory to your project.

h1. Usage

Import the category to your view controller and call registerShakeTrigger, passing in the block to execute when the user shakes the device.
